# Lawn 3D Visualization

A precise 3D web-based visualization of a residential property and its adjoining lawn, built using [Next.js](https://nextjs.org), [React Three Fiber](https://r3f.docs.pmnd.rs/), and [Drei](https://github.com/pmndrs/drei). 

This project aims to accurately render an architectural blueprint into an interactive 3D model, mapped down to the exact relative coordinate dimensions.

## Features

- **Architectural House Model**: A detailed structural model of the house consisting of an accurate L-shaped gallery traversing from the entrance porch all the way to the back. Fully aligned to the provided blueprints with inner walls, specific passage doors, and correctly sized room footprints.
- **Accurate Topography**: An elevated foundation plinth with dynamic block staircases descending to the lawn and the main street gate. 
- **Lawn Environment**: Includes a lush green lawn, perimeter boundary walls, planters, a fire pit, and BBQ area to accurately reflect the real-world property mapping. 

## Getting Started

First, install the dependencies if you haven't already:

```bash
npm install
```

Then, run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the 3D scene in action.
